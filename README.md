
Linux / macOS:
```
uv venv .venv
source .venv/bin/activate
uv pip install -e .
```

Windows (Command Prompt):
```
uv venv .venv
.venv\Scripts\activate
uv pip install -e .
```

Windows (PowerShell):
```
uv venv .venv
.venv\Scripts\Activate.ps1
uv pip install -e .
```
